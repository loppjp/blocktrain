# BlockTrain

Component building blocks for supervised ML training

# Quickstart

- use a recent python
    - `python3.12 -m venv env`
- get latest pip
    - `pip install -U pip`
- Install poetry
    - `pip install poetry`
- Install blocktrain
    - `poetry install`
